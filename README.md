# Bitcoin vs Gold – Data Analysis Project

This project analyzes Bitcoin’s historical performance and compares it with Gold to determine whether Bitcoin is a good investment. The analysis follows the Google Data Analytics Process: Ask, Prepare, Process, Analyze, Share, and Act.

##  Objectives
- Evaluate Bitcoin’s returns, volatility, and long-term growth.
- Compare Bitcoin’s performance with Gold.
- Identify trends, risks, and insights to support investment decisions.

##  Tools & Technologies
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

##  Dataset
Historical data for:
- **Bitcoin** (timestamp, open, high, low, close, volume)  
- **Gold** (date, price)

##  Key Analysis Performed
- Trend visualization (BTC vs Gold)
- Volatility comparison
- Rolling averages
- Correlation analysis
- Overall risk & return evaluation

##  Results
- Bitcoin shows high growth but higher volatility.
- Gold is more stable with lower risk.
- Investment choice depends on investor’s risk tolerance.
